# SecureDrop

CISC 468 Project – Secure Peer-to-Peer File Sharing Application

## Team Members
- Monica Stef (Go Client)
- Ben Leray (Python Client)

## Overview
SecureDrop is a peer-to-peer secure file sharing system that enables clients written in different programming languages to securely discover each other and exchange files over a network.

The system guarantees:
- Mutual authentication
- Confidentiality and integrity
- Perfect forward secrecy
- Secure local storage
- Key migration

## Project Structure
- `go-client/` – Go implementation
- `python-client/` – Python implementation
- `java-client/` – Java implementation
- `docs/` – Protocol specification and design
- `tests/` – Test cases
